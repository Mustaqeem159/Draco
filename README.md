## Why?

Every Monero wallet looks like it was built in 2005. 
Tarn is an attempt to bring modern UI/UX to 
the Monero ecosystem without sacrificing privacy.

## Goals

- 🔒 Full Monero node integration
- 🎨 Clean modern interface
- 👁️ Privacy by default — Tor/i2p support
- 💻 Desktop first (Linux, macOS, Windows)

## Status

> ⚠️ This project is in early development. 
> Not ready for real funds.
